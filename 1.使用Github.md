# 1.使用Github

## 1.1 目的

借助github托管项目代码

## 1.2 基本概念

1. **仓库（Repository）**
   即存放项目代码，每个项目对应一个仓库
2. **收藏（Star）**
3. **复制克隆项目（Fork）**
4. **发起请求（Pull Request） **
5. **关注（Watch）**
6. **事务卡片（Issue）**
7. **Github主页**
   显示用户动态及关注的用户或仓库的动态，右侧显示所有的git库
8. **仓库主页**
   显示项目信息，如源代码，版本，收藏/关注/fork情况等
9. **个人主页**
   个人信息，关注的人或git库，我的开源项目，我贡献的开源项目等
